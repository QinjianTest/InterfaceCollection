# 接口数据收集平台
- 平安寿险测试工具
- 本项目是一个接口数据收集平台，专门用于收集和分析JMeter自动化测试的结果数据，提供数据统计、分析和查询功能，让领导可以每天查询测试结果和性能指标

## 功能特性
- 自动收集JMeter测试结果
- 数据统计与分析
- 结果查询和过滤
- 可视化报告生成
- 用户管理和权限控制

## 技术栈

- **前端** ：HTML, CSS
- **后端** ：Python-Flask
- **数据库** ：Mysql
- **其他** ：JMeter

## 安装

### 克隆项目

```bash
git clone https://github.com/QinjianTest/InterfaceCollection.git
```

## 启动

```bash
python app.py
```
